Программный агент для выявления DeepFake в реальном времени

Курсовой проект по специальности «Кибербезопасность». Программный комплекс предназначен для автоматизированного выявления сгенерированного нейросетями медиаконтента (DeepFake) непосредственно в процессе воспроизведения видеопотока в веб-браузере пользователя.

## 🛠 Требования
* Python 3.9+
* pip
* Браузер на движке chromium

## Как запустить

1. Клонируйте репозиторий:
git clone https://github.com/Lodenerer/Kursovaya
cd repo

2. Установите зависимости:
Для работы проекта необходимы библиотеки
Установите их с помощью команды:

pip install -r requirements.txt

*(Примечание: необходимо подождать).*

3. Добавляете расширение на браузер
Создаёте папку с агентом "tiktok_agent_extension" и добавляете туда content.js и manifest.json
Заходите в браузере в chrome://extensions/ и через "Загрузить распакованное расширение" выбираете папку с агентом
Включаете его и проверяете есть ли оно в вашем списке расширений

4. Запускаем сервер
Запускаете main.py и ждёте пока запустится сервер

*(Примечание: необходимо подождать).*

6. Проверяем
После запуска сервера, переходим обратно в браузер, заходим на https://www.tiktok.com (В списке расширений, наше должно стать активным)
Листаем ленту рекомендаций, у видео должна появляться жёлтая рамка, меняющая цвет. Если цвет оранжевый - ошибка связи с сервером, остальные интуитивно понятны
